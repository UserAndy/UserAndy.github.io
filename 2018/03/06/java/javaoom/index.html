Linux ulimit 参数

  上面说到了，与系统(Linux)中的ulimit参数有关。ulimit 是用来限制系统资源的。
其中包括:

  1. max memory size (最大内存大小)。
  2. open files(打开文件数)。
  3. max user processes(最大用户进程数)
    等等。

系统性能优化常常会优化此参数。

(有兴趣的童鞋可以自行了解更多)
