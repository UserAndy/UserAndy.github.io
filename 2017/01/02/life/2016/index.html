2016年总结

发表于 2017-01-02   |   分类于 Life   |     |   阅读次数

我的2016年

2016年，我认为是一个比较失败的一年，
在这一年中，我浪费了不少机会，也失去了不少机会，

失去&不足：

  1. 在有多个选择时，有做出好的选择.(ps: 工作，生活均有)
  2. 做事情有些纠结，执行力不够，
  3. 做事情效率低,主要原因是方法，思维不正确，不专一而致。
  4. 没能坚持每周分享一篇博客，

收获:

在过去的2016年，也有一些成长，

  1. 获得了驾照
  2. 开始写原创博客，
  3. 开源了自己的第一个开源项目.
  4. 开始看书，涉及专业以及不专业的书籍(以下是2016年的书单)

    《淘宝技术这十年》
    《逻辑思维：成大事这不纠结》
    《微信帝国》

小结:

在过去的2016年里，经历了不少事情，主要从生活，技术这两方面来进行总结.
生活上：

  1. 在面对选择的时候，眼光应该往后看几年，应该更加注重以后的发展,(如:在之前的offer选择时，就因为一点薪资，浪费了一些比较好的机会，)
  2. 不与别人进行薪资上的攀比，盲目的攀比只能让自己陷于尴尬的局面
  3. 沟通能力不强，一些自己知道，理解的东西却不能通过更加清晰的思路表达出来，

技术上：

  1. 不专一，什么多想学，执行力却不够
  2. 没有看一本专业书籍，对一些基础知识掌握不扎实
  3. 技术上没有新的突破

总结: 对过去的2016年是不满意的，用几个字来概括的话就是，’纠结’，‘不专一’，‘选择错误’。

在2017年，我相信会更好，同时也会加大力度维护公众号，博客。坚持分享!

PS:祝大家2017年新年快乐!
