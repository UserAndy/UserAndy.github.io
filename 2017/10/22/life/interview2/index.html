面试官初体验

发表于 2017-10-22   |   分类于 Life ， 面试   |     |   阅读次数

前言

  由于公司业务发展需要,最近有大量的招聘需求,我也有幸成为面试官，进行了一系列的面试，其中大部分都是两年左右工作经验的求职者，从中发现不少问题，也有一些初次做面试官的体会，记录在这里，也算是第一次做面试官的总结吧，如果能够帮助到你,那就更好了!

简历

  大家都知道，求职时，一份好的简历，至关重要！不仅能让你获得HR的面试邀请，也能让你得到面试官的青睐，在最近的面试时，发现非常多求职者的简历是不合格的，为什么不合格呢？因为大部分的求职者的简历要么内容臃肿，要么太简单，要么错别字太多，比如：

有将SVN写成SVM的，有将fastjson写成FashJson的，有将SpringMVC写成SpringMvc的,有将MySQL写成Mysql的……等等
简历上有错别字的，太多太多了! 更有甚者,在简历中的个人技能页上:专有词汇出现7,8个错别字。这样的简历,面试官看到的第一感觉,就是不专业。

态度

  这个是最基本的要求,无论是经验丰富的同学，还是刚毕业的新同学,在求职的时候，都是需要注意的,在最近的面试中，大部分求职者的态度都是挺好的,但有部分同学，在面试是,态度就有些欠缺，请看下面两段对话:
对话一:

面试官:你对Git熟悉吗？
求职者A: 这个不太熟悉,平时SVN用的多些,Git也不是高深的技术,网上一搜一大把。
面试官: …..

对话二:

面试官: 你对Git熟悉吗？
求职者B: 这个不太熟悉,平时SVN用的多些,这方面的知识确实有些欠缺,我会在工作之余多了解的。
面试官: 嗯，好的

看到这里,你可能在笑，在最近的面试过程中，像职者A这样的还不少呢!(我觉得求职者A说的确实是对的,但这样的对话,就已经体现出求职者的态度了)。

沟通能力

  这个并不是说求职者,对我自己也同样适用,在沟通方面,也同样有欠缺,这个算不算是程序员的通病呀(微笑脸)。
就比如我：在之前的求职过程中,面试官问的问题,心里知道,工作中也用过,但就是表达不出来,表达的不清楚, 其实不仅仅在面试方面，在需求讨论,bug描述等等，都和沟通能力有很大的关系。我也有待提高!

自信

  在最近的面试过程中,有些求职者的技术还是蛮不错的,但是在面谈的过程中，其实还是比较紧张的,自己平时知道的知识点，说起来就没那么顺畅,有点支支吾吾的感觉,更有甚者，还没开始面试的时候，就说自己技术能力不行,什么什么没做过,这些都是不太好的，既然来面试了，就要自信一点，相信自己的能力,即使能力不够,也能起到一个查漏补缺的作用。
ps: 自己之前在求职过程中,也是非常的不自信。

技术能力

  此次面试时,发现很多求职者的基础还是不够的,问Java SE 方面的知识,答的时候都挺支支吾吾的，一些基础的数据结构也说不清楚,还有一些小伙伴，使用框架时间久了，对基础就不了解了，比如:

说说jdbc连接数据库常用的几个java类
都有些童鞋说不清楚，对常用的集合类,网络类都不了解，这就很不应该呀。

对求职者的建议

简历

首先我们来看看简历是什么:

其实对于我们软件行业,简历就是通过简洁的描述来介绍自己,所掌握的技能,项目经验等等。

我个人认为:
简历=自我介绍+所掌握的技能列表+项目经验
从面试官的角度,我认为一份合格的简历，至少要符合以下三点:

  1. 够简洁,(不是简单)，但要表达清楚，比如有几年工作经验后，写项目时，选择比较有代表性的项目，描述清楚，而不是每个项目都写上。（个人建议最多2页）
  2. 专业词汇必须写正确，大小写也应该严格区分！比如fastjson不能写成FaSTjson，（不然会让面试官的感觉求职者很不专业！）
  3. 项目经验至少要详细介绍1至2个项目,其中包括项目简介,项目起始时间,自己所承担的职责以及所负责的模块,项目收获等。
  4. 关于简历格式方面,个人强烈建议使用pdf格式。

态度&自信

  1. 态度方面,当面试官问的一些自己不知道的问题时，不清楚的时候，态度要端正,表明自己不清楚，后续会花时间了解,(ps: 每个人的知识面,技能点都是不一样的,不清楚很正常,但是不清楚,态度还不好,还不肯自己花时间去思考,那就不正常了)。
  2. 面试时,不要紧张,自己知道多少,就说多少,想想之前自己面试的时候,也挺紧张的，估计这次换位体验后，应该会好很多了。
  3. 面试的时候，要相信自己,不要怯场,你表达出自己的能力后就足够了。更不要花了好几个小时来准备面试,到了公司,坐在面谈室时,说自己技术不好,技术怎么差，这个时候，自己的技术能力好不好，就让面试官来评价好了。

小结

  上面说了这么多,有些方面自己也还是有些欠缺的,在这之前,都是以求职者的身份面试,这次有机会来面试别人,其实还是挺幸运的,在面谈的过程中,看求职者回答问题的时候，我就在想，这个问题，我会怎么回答，如何答的更好,更全面等等。此次面试官体验，也是对自己的一个查漏补缺的过程，对自己也还是挺有帮助的。

最后:  祝大家都能找到一个满意的工作！
