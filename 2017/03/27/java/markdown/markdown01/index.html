Markdown详解(一)

1.1 Markdown简介

Markdown是一种轻量级标记语言,创始人为约翰·格鲁伯(John Grruber)。它允许人们”使用易读易写的纯文本格式编写文档,然后转换成有效的XHTML（或者HTML)文档。吸收了很多在电子邮件中已有的纯文本标记的特性。广泛应用于:Stack Overflow,GitHub,Bitbucket等网站,常见的个人博客也大量使用Markdown。
