Android 集成科大讯飞语言SDK(一)

发表于 2016-06-17   |   分类于 Android   |     |   阅读次数

现在项目中，经常需要集成语言识别SDK,常用的语音识别的SDK有:百度语音识别，科大讯飞语音识别 今天为大家介绍科大讯飞语音识别SDK的使用

1.1 准备工作

在使用之前，我们需要在其官网下载官方提供的SDK,下载时，需要注册科大讯飞的账号，这里不再赘述,注册完成后，进入首页如图所示:

1.2 创建应用

到这里,我们点击控制台创建应用，如图所示:

进入控制台后,点击创建应用,填写一些必要的应用名称，应用分类，应用描述，选择应用平台。
如图所示:

点击创建即可，创建完成后，我们点击左侧我的应用选项，查看刚才创建应用。
如下图所示:

1.3 开通所需服务

通过步骤1.2 的创建应用后，该应用还没开通任何服务，现在我们点击立即开通，如下图所示:

开通后如下图所示:

1.4 下载 SDK

我们开通服务后,已经完成一大半了，现在需要做的是下载sdk,点击左侧sdk下载，如下图所示:

跳转至下图页面:

在这里，我们可以选择单个服务SDK下载或者组合服务SDK下载，然后选择服务，选择对应的平台后，选择应用，即可下载SDK,
